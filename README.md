## 📌 Project Overview
This project is a Computer Vision application designed to automate person counting in public spaces like malls. It uses **YOLOv8** for real-time detection and **ByteTrack** to maintain unique IDs for every person, allowing us to track their direction of movement.

### The Problem
Manual counting of people entering or exiting a building is inefficient and prone to human error. This project provides an automated way to monitor crowd flow for safety and business analytics.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **AI Model:** YOLOv8 (yolov8s.pt)
* **Libraries:** * `ultralytics` (for YOLOv8 & Tracking)
  * `opencv-python` (for Video Processing)
  * `matplotlib` (for Visualization)
* **Dataset:** [Mall Dataset from Kaggle](https://www.kaggle.com/datasets/chaozhuang/mall-dataset)
